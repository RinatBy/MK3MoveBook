MK3 MOVEBOOK

1. Распакуйте архив в отдельную папку.
2. Не переносите отдельный EXE: папка web и DLL-файлы должны лежать рядом.
3. Запустите «MK3 MoveBook.exe» двойным щелчком.

Если WebView2 недоступен, запустите «Web MK3 MBook.exe». Книга
откроется в браузере, выбранном в Windows по умолчанию.

Интернет нужен только для проверки и
загрузки обновлений через кнопку «Обновления».

Если программа сообщает об отсутствии WebView2, установите Microsoft Edge
WebView2 Runtime. Программа предложит запустить официальный установщик
Microsoft из этого архива. Для загрузки Runtime потребуется интернет.

В веб-версии автоматическая установка обновлений недоступна. Кнопка
«Обновления» откроет страницу последнего выпуска для ручного скачивания.
